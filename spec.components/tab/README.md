<div class="mb40">
    <div class="fontsize-20">标签切换 <small>Tab</small></div>
    <div class="color-999 mt4">TAB是通过折叠信息紧凑展示页面内容，避免在有限的显示区域内展示过多信息的一种方式。根据信息和内容本身的属性，以及用户任务，对所有的信息和内容进行分类，而不减少网页的信息容量。</div>
</div>
