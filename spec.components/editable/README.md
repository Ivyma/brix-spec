<div class="mb40">
    <div class="fontsize-20">就地编辑 <small>Editable</small></div>
    <div class="color-999 mt4">用户可以就在当前页面内容所处的地方直接编辑该内容，而无须跳转到另一个新页面去编辑。</div>
</div>
